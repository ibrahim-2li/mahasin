# لوحة التحكم - محاسن الشرق

## تفعيل لوحة التحكم

### 1. تشغيل Migration لإنشاء جدول قاعدة البيانات

افتح Terminal في مجلد المشروع وقم بتشغيل الأمر التالي:

```bash
php artisan migrate
```

### 2. إضافة البيانات الافتراضية (اختياري)

```bash
php artisan db:seed --class=SiteContentSeeder
```

### 3. الوصول إلى لوحة التحكم

بعد تشغيل السيرفر، يمكنك الوصول إلى لوحة التحكم عبر الرابط:

```
http://127.0.0.1:8000/admin/dashboard
```

## كيفية الاستخدام

1. **تعديل النصوص**: 
   - قم بتعديل النص في الحقل المطلوب
   - اضغط على زر "حفظ"
   - ستظهر رسالة تأكيد باللون الأخضر

2. **رفع الصور**:
   - اختر الصورة من جهازك
   - اضغط على زر "رفع الصورة"
   - سيتم تحديث الصفحة تلقائياً لعرض الصورة الجديدة

3. **معاينة التغييرات**:
   - اضغط على "عرض الموقع" في الأعلى
   - ستفتح نافذة جديدة لمعاينة الموقع مع التغييرات

## ملاحظات مهمة

- جميع الصور يتم رفعها إلى مجلد `public/images`
- الحد الأقصى لحجم الصورة: 2MB
- الصيغ المدعومة: JPG, PNG, GIF
- جميع التغييرات تحفظ مباشرة في قاعدة البيانات

## استكشاف الأخطاء

إذا واجهت مشكلة في رفع الصور، تأكد من:
1. صلاحيات الكتابة على مجلد `public/images`
2. حجم الصورة لا يتجاوز 2MB
3. صيغة الصورة مدعومة

## الأمان

⚠️ **مهم**: لوحة التحكم حالياً غير محمية بنظام تسجيل دخول. 
يُنصح بإضافة نظام مصادقة (Authentication) قبل النشر على السيرفر الفعلي.

لإضافة حماية سريعة، يمكنك استخدام Laravel Middleware أو إضافة كلمة مرور بسيطة.
